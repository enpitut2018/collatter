# Colatter ~ コラッター
**コラ作成＆共有アプリ**

## エレベーターピッチ
手軽にコラ画像を作成し、共有することができる「Colatter」というアプリです。元画像の吹き出しの白塗りや、文字挿入、Twitterへの投稿が一括でできます。また、作成したコラ画像のテンプレートをColatter上に投稿することができ、他のユーザーが作成したテンプレートからコラ画像を作成することもできます。
